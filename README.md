Datapack + Resource Pack that adds tool trims (uses the same templates & resources as armour trims)

Crafting is also the same as armour trims.

Currently:
* Pickaxe
* Axe
* Shovel
* Hoe
* Sword
* Bow
* Crossbow

Planned (Whenever I get a around to making texture for them):

* Trident
* Mace
